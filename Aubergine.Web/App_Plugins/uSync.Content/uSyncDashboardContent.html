uSync.ContentEdition is installed. Content and Media Handlers can now import and export content, as part of a standard sync.

uSync.ContentEdition

uSync.ContentEdition, adds handlers to the existing uSync Imports.

There is nothing to configure with Content Edition, but you will see two new handlers in the config section.

  • uSync: ContentHandler
  • uSync: MediaHandler

you can if you want turn these on and off in the uSyncBackoffice.config file.

Content Mappers

Content mappers handle the mapping of content ids inside of properties, mappers will be called when a property is of a certain type, they translate and internally stored Ids into more global Ids (usally GUIDS) so things like links work across sites.

Content Mappers are configured in uSyncCore.config
